# 🏥 MboaMed – Reimagining Healthcare Access in Cameroon

> **"Faster Care. Shorter Waits. Better Lives."**

MboaMed is a modern digital healthcare platform designed to improve access to medical services across Cameroon. The project aims to reduce waiting times, simplify appointment scheduling, help patients locate nearby hospitals and pharmacies, and provide a more connected healthcare experience for both patients and healthcare providers.

🔗 **Live Demo:** https://medi-fast-cameroon-deploy-three.vercel.app/

---

# 🌍 Why MboaMed?

Accessing healthcare in many parts of Cameroon remains unnecessarily difficult.

Patients often spend hours waiting at hospitals without knowing doctor availability, struggle to locate nearby healthcare facilities during emergencies, and have limited access to reliable digital medical services.

MboaMed was created as an exploration of how modern web technologies and AI-assisted software development can contribute toward solving these real-world challenges.

Rather than replacing healthcare professionals, the platform focuses on making healthcare **more accessible, organized, and patient-centered.**

# 💡 Project Vision

Healthcare should not depend on long queues, fragmented information, or geographical barriers.

MboaMed explores how digital technologies can help create a healthcare ecosystem where patients can quickly connect with hospitals, doctors, and essential services from a single platform.

The long-term vision includes AI-assisted healthcare guidance, secure medical record management, telemedicine integration, and intelligent appointment optimization.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Gitmitty/MboaMed.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Create production build

```bash
npm run build
```
